# HTML

## 行内元素 块级元素 空元素

行内元素：

`<a>`：用于创建超链接。

`<span>`：用于为文本或其他元素设定样式。

`<strong>`：表示强调文本。

`<em>`：表示斜体或强调文本。

`<img>`：用于插入图像。

`<input>`：用于创建表单输入控件等。

块级元素：

`<div>`：用于组合其他元素或划分页面区块。

`<p>`：表示段落。

`<h1>` - `<h6>`：表示标题。

`<ul>` 和 `<ol>`：用于创建无序列表和有序列表。

`<table>`：用于创建表格等。

空元素：

`<br>`：用于换行。

`<hr>`：用于水平分隔线。

`<img>`：用于插入图像。

`<input>`：用于创建表单输入控件等。
